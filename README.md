## Installation

```
git clone https://github.com/guruprasadhj/Tuby
cd Tuby
pip install .
```
 ## Run the App 
 ### GUI :
 ```
 tuby-gui  
 ```
 ### CLI :
```
 tuby-cli
```
_____
## Dependency to Run Code

### packages
- python-3.x
- python-tk
- xsel or xclip
### Library
- pytube
- PIL(pillow)
- PyperClip
