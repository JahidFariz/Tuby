## Installation

```
git clone https://github.com/guruprasadhj/Tuby
cd Tuby
pip install requirements.txt
pip install .
```
 ## Run the App 
 ### GUI :
 ```
 tuby-gui  
 ```
 ### CLI :
```
 tuby-cli
```
